# Tilapia Tank Monitoring

A low-cost Internet of Things (IoT)-based system for continuous water quality monitoring in tilapia farming.

## Overview

This project presents a low-cost monitoring system designed for aquaculture environments. The system continuously collects water quality parameters using multiple sensors connected to an ESP8266 (ESP-12E) microcontroller and sends the collected data to the ThingSpeak cloud platform for remote visualization and analysis.

The project was developed as part of a research study on IoT-based water quality monitoring for tilapia farming.

---

## Features

- Real-time water quality monitoring
- Wi-Fi communication using ESP8266
- Cloud data storage with ThingSpeak
- Low-cost hardware
- Continuous monitoring
- Sensor calibration support

---

## Monitored Parameters

- Water pH
- Water Temperature
- Turbidity
- Total Dissolved Solids (TDS)
- Oxidation-Reduction Potential (ORP)
- Environmental Gas (MQ135)

---

## Hardware

- ESP-12E (ESP8266)
- SEN0161-V2 pH Sensor
- DS18B20 Temperature Sensor
- TS100 Turbidity Sensor
- SEN08030 TDS Sensor
- ORP 5012 Sensor
- MQ135 Gas Sensor
